Circuito arduino: Blink

Este circuito me sirvió mucho para comenzar a entender mejor las funciones de cada comando y a tener una nueva aproximaci’on con Arduino. Probé el código primero en TinkerCad para poder ver bien primero como y donde eran las conexiones. Básicamente el circuito entrega infomración a una Led para que se prenda y apague con un delay que en este caso es de 1 segundo, manteniéndose encendida durante 2 segundos.

El código es:

void setup()

{ pinMode(13, OUTPUT); }

void loop() { digitalWrite(13, HIGH);

delay(2000);

digitalWrite(13, LOW);

delay(1000); }

Circuito arduino: Potenciometro

El potenciómetro básicamente es el regulador que permite la cantidad de energía que pasa para poder alimentar la Led.

Código:

int sensorPin = 0;

int ledPin = 13;

void setup() {

pinMode(ledPin, OUTPUT); }

void loop() {

int sensorValue;

sensorValue = analogRead(sensorPin);

digitalWrite (ledPin, HIGH);

delay (sensorValue);

digitalWrite (ledPin, LOW);

delay (sensorValue); }

Circuito arduino: RGB

Por primera vez trabjé con una Led que cambiara de colores, y siendo sincero, aún no entiendo mu ybien como, solo se que cada entrada le otorga una instrucción de seguir un color, pero no entiendo muy bien cuantos colores puede iluminar.

Código:

const int RED_PIN = 9;

const int GREEN_PIN = 10;

const int BLUE_PIN = 11;

int DISPLAY_TIME = 100;

void setup() {

pinMode(RED_PIN, OUTPUT);

pinMode(GREEN_PIN, OUTPUT);

pinMode(BLUE_PIN, OUTPUT);

}

void loop() {

digitalWrite(RED_PIN, HIGH);

digitalWrite(GREEN_PIN, LOW);

digitalWrite(BLUE_PIN, LOW);

delay (1000);

digitalWrite(RED_PIN, LOW);

digitalWrite(GREEN_PIN, HIGH);

digitalWrite(BLUE_PIN, LOW);

delay(1000);

digitalWrite(RED_PIN, LOW);

digitalWrite(GREEN_PIN, LOW);

digitalWrite(BLUE_PIN, HIGH);

delay (1000); }

#include <Servo.h> Servo myservo;

int value; double angle;

void setup()

{ Serial.begin(9600); myservo.attach(2); }

void loop()

{ value = analogRead(A0); angle = map(value, 0, 1023, 0, 180); Serial.println(angle); myservo.write(angle);
delay(15);
}
